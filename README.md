# bionano-lib
Package for BioNano data manipulation.

**Currently supported formats:**
* BNX
* MAP formats - CMAP, XMAP, SMAP and MAP
* CN
* ERR
* MAPRATE
* BED

**Supported languages:**
* R 3.4.x
* Python 3.x - Coming soon
